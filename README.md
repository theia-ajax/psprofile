
# PSProfile

Config files for PowerShell, Wezterm, and Starship.rs for a more complete, customizable, and faster terminal.

## Prereqs

- Powershell 7.5+ `winget install --id Microsoft.Powershell --source winget`
- Wezterm `https://wezterm.org/installation.html`
- Starship.rs `https://starship.rs/guide/#%F0%9F%9A%80-installation`

## Installation

- Install Fira Mono NerdFont(s) from FiraMono.zip
- Copy the config files to their appropriate location
    - `user_profile.ps1` -> `%USERPROFILE%\Documents\Powershell\Microsoft.Powershell_profile.ps1`
    - `wezterm.lua` -> `%USERPROFILE%\.wezterm.lua`
    - `starship.toml` -> `%USERPROFILE%\.config\starship.toml`
